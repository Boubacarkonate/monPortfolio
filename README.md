# Portfolio

Ce dépôt contient mon portfolio personnel, créé pour présenter mes compétences, mes projets et mon parcours professionnel. Le site est développé avec une structure statique légère : **HTML**, **CSS** et **JavaScript**.

## Présentation

Mon portfolio présente :
- une introduction personnelle
- une section compétences
- une galerie de projets
- une section formations
- un contact avec liens GitHub/LinkedIn et email

## Technologies

- HTML5
- CSS3
- JavaScript
- Police Google Fonts
- Icônes Font Awesome

## Fonctionnalités principales

- Switch de langue **FR / EN** pour le texte du site
- Menu hamburger mobile responsive
- Effet de défilement et animations de révélation au scroll
- Canvas de particules animé en arrière-plan
- Navigation active basée sur l’intersection des sections
- Boutons de projets et liens externes

## Structure du projet

- `index.html` — page principale du portfolio
- `style.css` — styles globaux et responsive
- `script.js` — logique JavaScript pour l’interface et l’internationalisation
- `public/lang_fr.json` — contenus français
- `public/lang_en.json` — contenus anglais
- `public/media/` — images et ressources statiques


## Notes

- La traduction est gérée côté client via JSON.
